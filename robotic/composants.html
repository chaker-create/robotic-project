Les 3 Composants Principaux

Pour qu'un robot fonctionne, il a besoin de trois parties essentielles qui travaillent ensemble :

  • 1. Les Capteurs (Sensors) : Ce sont les "yeux" et les "oreilles" du robot. Ils lui permettent de collecter des informations sur le monde extérieur (température, lumière, distance, image).
  • 2. Le Cerveau (Processeur/Algorithme) : C'est l'ordinateur embarqué. Il reçoit les informations des capteurs, les analyse et prend une décision selon le programme installé.
  • 3. Les Effecteurs (Actionneurs) : Ce sont les "muscules" du robot. Ils reçoivent l'ordre du cerveau pour agir physiquement (moteurs pour rouler, bras pour saisir un objet).
